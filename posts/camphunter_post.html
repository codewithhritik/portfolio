Camphunter

Click here to visit the page

This was a project I started back in February 2020 while learning web development. I am the sole creator/developer for this project.

This is a Full stack application with user authentication and authorization for log in/sign up.

In this project, after logging in, a user can add, delete, as well as update their campgrounds. Each campground can have comments from various users across the platform. You can view the campground of your choice, locate it and get prices for the same!

Technologies:
  • - HTML/CSS
  • - JavaScript
  • - Bootstrap
  • - Node JS
  • - Express JS
  • - MongoDB
  • - Git
  • - Heroku
