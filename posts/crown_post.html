This was a project I started back in December 2020 while learning React. I am the sole creator/developer for this project.

This is a Full stack application with user authentication and authorization for log in/sign up using Firebase.

In this project, you will be able to shop clothing online like amazon, flipkart, alibaba or any other e-commerce website with payment integartion using stripe.

Technologies:
  • - HTML/CSS
  • - JavaScript
  • - React
  • - Redux
  • - Styled-Components
  • - Firebase
  • - Stripe API(for payments)
  • - Git
  • - Heroku
  • - Persistance + Session Storage
